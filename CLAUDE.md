# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Type

This is an Obsidian vault (personal knowledge management system), not a software project. There is no build system, tests, or traditional code architecture.

## Obsidian Vault Structure

- `.obsidian/` - Vault configuration, plugins, and workspace state
- `.obsidian/plugins/` - Community plugins (obsidian-git, claudian, terminal)
- `.claudian/` - Claude Code session data and settings
- `*.md` - Markdown notes (the actual content)

## Key Notes

- Main topics documented: HDC (High Dimension Computing), 神经记录技术 (Neural Recording Technology), paper outlines
- Backups are managed via obsidian-git plugin
